# 深度学习函数拟合实验：纯NumPy与TensorFlow框架的双轨对比

本项目旨在通过构建并训练基于ReLU激活函数的多层感知机（MLP），对特定的混合非线性连续函数进行高度逼近拟合。实验采取了双轨并行的策略：分别使用现代深度学习框架（TensorFlow/Keras）与纯底层矩阵运算库（NumPy）来实现完全相同的网络拓扑结构，以此直观对比框架抽象引擎与底层数学推演在模型收敛、精度及性能上的差异。

## 实验原理与目标

* **目标函数**：y = sin(x) + 0.5x
    该函数融合了三角函数的高频非线性周期震荡特征与一次函数的全局稳定线性趋势，极其严苛地检验了神经网络模型对复杂混合数据分布的特征提取能力。
* **网络拓扑架构**：
    * 输入层：1个神经元（接收自变量 $x$）
    * 隐藏层1：64个神经元，使用 ReLU 激活函数
    * 隐藏层2：64个神经元，使用 ReLU 激活函数
    * 输出层：1个神经元，无激活函数（线性输出）
* **损失函数**：均方误差（MSE）

## 项目结构

项目解耦为数据生成、NumPy底层实现与TensorFlow框架实现三个独立模块，以确保实验逻辑的清晰与对比的绝对公平。

* `generateDataset.py`：数据采集与预处理模块。基于规范，生成1250个全量样本，并使用严格的无放回抽样互斥切分为训练集（80%）与测试集（20%），最终输出至 `data/` 目录。
* `numpyStructure.py`：纯底层手工网络模块。完全依靠纯NumPy的矩阵乘加运算构建前向传播，并基于微积分链式法则手工推演偏导数，使用静态学习率的批量梯度下降（BGD）完成参数更新。
* `tensorflowStructure.py`：深度学习框架模块。依靠 `tf.keras` 构建计算图，利用底层自动求导机制（Autodiff）与自适应的 Adam 优化器进行高效的模型训练。

*(运行代码后会自动生成以下目录)*
* `data/`：存放生成的 `train.csv` 与 `test.csv`。
* `results/`：存放模型训练过程的 Loss 数据、预测结果以及最终带有95%统计置信区间的对比可视化图表（`.png`）。

## 实验结论与对比分析
### 收敛动力学差异：
得益于带有动量加速与自适应学习率调节功能的 Adam 优化器，TensorFlow 模型在极短的迭代内即可完成收敛；而纯NumPy手工模型由于依赖基础的批量梯度下降（BGD），需历经数千次迭代才能跨越误差曲面的鞍点与平原，达到相近的稳态。

### 拟合精度与视觉特征：
两种路径均完美重构了复杂的非线性函数。在微观层面，TensorFlow 预测曲线展现出极致的平滑度；NumPy 预测曲线则因优化器的局限，在局部高曲率区域微弱地保留了 ReLU 激活函数“分段线性逼近”的物理拼接痕迹。

### 量化评估：
在完全隔离的测试集上，两种方案的均方误差（MSE）均降至极低量级，彻底排除了过拟合的可能。其中，TensorFlow凭借框架优化优势实现了更低的精度。
